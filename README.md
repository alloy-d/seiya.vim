seiya.vim
=========

For you and your "waifu"

## Usage

#### Enable

Execute ``:SeiyaEnable`.

#### Disable

Execute ``:SeiyaDisable`.

If backgroung color is not updated, try `colorshame [your_colorscheme]`

#### Automatically enable when vim starts

Add this line in your vimrc.

```vim
let g:seiya_auto_enable=1
```
